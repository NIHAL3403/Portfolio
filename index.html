Project 1: Motion Detection and Remote Monitoring System

Issuing Organization: Reva University
Application number: 202441011575 (Submitted Patent)

Components:

ESP32-CAM Microcontroller: A low-cost development board with an integrated camera, ideal for capturing images and videos.
PIR Sensor: A Passive Infrared sensor that detects motion by measuring infrared radiation changes in its environment.

Functionality:

Motion Detection: The PIR sensor detects motion and triggers the ESP32-CAM.
Image/Video Capture: The ESP32-CAM lights up and captures images or records videos when motion is detected.
Notification System: Sends a notice to the owner when motion is detected.
Web-Based Interface: Owners can monitor their premises remotely through a web interface, provided they have internet access.
Applications: Home security systems, remote surveillance, wildlife monitoring, and more.
Description: The system incorporates an ESP32-CAM microcontroller together with a PIR sensor. The ESP32-CAM lights when motion is detected and takes pictures or records videos with a notice sent to the owner. The web-based interface allows owners to monitor their premises from anywhere with internet access.

Project 3: Face Detection and Emotional Analysis

Description:

Facial recognition and analysis is a powerful vision tool that can read facial expressions in photos and videos. It takes place in two stages. It uses advanced algorithms to find the right face by first identifying facial features such as eyes and nose. It then analyzes the position of these features, especially the eyebrows, mouth and eyes, to interpret the emotions expressed.
Recognition:
Stage 1: Identifies key facial features such as eyes, nose, mouth, etc.
Stage 2: Analyzes the position and movement of these features to interpret the emotions expressed.
Algorithms and Techniques:
Feature Detection: Uses advanced algorithms to accurately identify facial features.
Emotion Analysis: Analyzes the position of features like eyebrows, mouth, and eyes to determine the emotional state (e.g., happy, sad, angry).
Applications: Security and surveillance, human-computer interaction, market research (understanding customer emotions), and healthcare (detecting emotional distress).

Project 4: Creation of News Application

Description:

The News Application is a modern, responsive web application that fetches and displays the latest news headlines from various sources. This project is built using React for the frontend and Node.js with Express for the backend. It utilizes the News API to retrieve real-time news updates and presents them in a user-friendly interface.

Key Features:


Fetching and Displaying News Headlines: The application fetches the latest news headlines from the News API and displays them in a clean and organized manner. Responsive Design: The application is designed to be responsive and mobile-friendly, ensuring a seamless user experience across different devices and screen sizes. User-friendly Error Handling: The application provides clear and informative error messages to the user in case of any issues, such as network errors or API failures. Backend Caching: To reduce the number of API calls and improve performance, the application implements caching on the backend. This allows for faster response times and reduces the load on the News API.
